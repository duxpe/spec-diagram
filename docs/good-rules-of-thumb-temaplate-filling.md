A few rules I’d apply to the exporter when filling the template:

Omit sections that are completely empty instead of printing placeholder noise.

Never invent missing information.

Keep the document centered on the selected N1 root block and include only relevant project-wide context.

Pull N2 and N3 details only from descendants of that N1 block.

Treat notes as supporting context, not as primary structure.

Prefer concise bullets over long prose when the source data is fragmented.
